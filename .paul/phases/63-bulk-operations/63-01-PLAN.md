<objective>
## Goal
Add batch commission approval API endpoint and multi-select checkbox UI on the payroll tab so payroll staff can approve commissions and mark entries paid in bulk instead of one-by-one.

## Purpose
Payroll staff currently click approve/mark-paid individually for each of 18 agents' weekly entries. Batch operations reduce this from dozens of clicks to select-all + one confirm per agent, directly cutting payroll processing time.

## Output
- New POST `/api/payroll/batch-approve-commission` endpoint
- Checkbox column in payroll WeekSection sale rows
- "Select All" toggle per agent card
- Floating action bar with batch approve + batch mark-paid buttons
- ConfirmModal summary before execution, single toast result
</objective>
