<objective>
## Goal
Rework CS tracking cards to show contact attempt count badges, expanded working area with log buttons + attempt timeline + resolve section, and soft gate override UI.

## Purpose
CS agents currently click "Resolve" with no accountability trail. This plan transforms the card expand from a simple resolve form into a full outreach workspace — log attempts, see timeline, then resolve — with the 3-call gate enforced visually and overridable with justification.

## Output
- API: `bypassReason` field on both resolve endpoints to allow gate override
- API: `notes` required on contact attempt creation (`.min(1)`)
- UI: Call count badge ("0/3 Calls") on each unresolved card row, green at 3/3
- UI: Expand trigger changed from "Resolve" to "Work" (broader scope)
- UI: Expanded area contains: Log Call/Email/Text buttons with required notes → attempt timeline → resolve section
- UI: "Override 3-call gate" checkbox with required justification textarea when resolving with < 3 calls
</objective>
