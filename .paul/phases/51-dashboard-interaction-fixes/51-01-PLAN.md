<objective>
## Goal
Replace all 9 `window.confirm()` calls with the themed ConfirmModal from @ops/ui, and add confirmation dialogs to 5 unprotected destructive actions — eliminating browser-native dialogs from the platform entirely.

## Purpose
Browser-native `window.confirm()` breaks the dark glassmorphism theme and provides no loading feedback for async operations. The ConfirmModal built in Phase 50 provides themed styling, async loading states, and WCAG-compliant accessibility. This plan also protects 5 destructive actions (unresolve, delete, unapprove, remove rep) that currently fire on a single click with no confirmation.

## Output
- Zero `window.confirm()` calls remaining in the codebase
- 14 total confirmation points using ConfirmModal (9 replaced + 5 new)
- All confirmations use danger variant for destructive actions, primary for neutral
</objective>

    **Pattern to apply in each file:**

    1. Import ConfirmModal: `import { ConfirmModal } from "@ops/ui";`
    2. Add state for modal management:
       ```
       const [confirmState, setConfirmState] = useState<{
         open: boolean;
         title: string;
         message: string;
         variant: "primary" | "danger";
         confirmLabel: string;
         loading: boolean;
         onConfirm: () => Promise<void> | void;
       }>({ open: false, title: "", message: "", variant: "primary", confirmLabel: "Confirm", loading: false, onConfirm: () => {} });
       ```
    3. Add helper to open modal: <!-- audit-added: confirmLabel parameter -->
       ```
       function requestConfirm(title: string, message: string, variant: "primary" | "danger", confirmLabel: string, action: () => Promise<void> | void) {
         setConfirmState({ open: true, title, message, variant, confirmLabel, loading: false, onConfirm: action });
       }
       ```
    4. Add handler for confirm:
       ```
       async function handleConfirm() {
         setConfirmState(s => ({ ...s, loading: true }));
         try { await confirmState.onConfirm(); } finally { setConfirmState(s => ({ ...s, open: false, loading: false })); }
       }
       ```
    5. Render ConfirmModal once at component bottom:
       ```
       <ConfirmModal open={confirmState.open} title={confirmState.title} message={confirmState.message} variant={confirmState.variant} confirmLabel={confirmState.confirmLabel} loading={confirmState.loading} onConfirm={handleConfirm} onCancel={() => setConfirmState(s => ({ ...s, open: false }))} />
       ```
