<objective>
## Goal
Migrate hardcoded hex colors and rgba() patterns to semanticColors + colorAlpha tokens, and hardcoded fontSize values to typography.sizes references — eliminating visual inconsistency across all dashboard files.

## Purpose
The dashboard has ~222 hardcoded color instances (30 unique hex values) and ~362 hardcoded fontSize values scattered across 20+ files. Plan 52-01 created the token system (semanticColors, colorAlpha, typography.sizes). This plan performs the bulk migration so all visual values reference centralized tokens.

## Output
- Hardcoded hex colors replaced with semanticColors.* references
- Hardcoded rgba() replaced with colorAlpha(semanticColors.*, alpha)
- Hardcoded fontSize values replaced with typography.sizes.*.fontSize
- Zero visual changes — tokens map 1:1 to existing values
</objective>
