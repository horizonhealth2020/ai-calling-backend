## Critical Gaps

### GAP-1: EH-04 — Socket.IO try/catch wrappers stripped
**Root cause:** Phase 28 executor ran in a worktree. Its changes to `socket.ts` (replacing `any` types) were computed against the pre-Phase-27 base. When merged, the diff removed the try/catch wrappers Phase 27 had added.
**Fix:** Restore try/catch wrappers in all 10 emit functions in `apps/ops-api/src/socket.ts`.

### GAP-2: EH-02 — Zod validation stripped from 8 route files
**Root cause:** Same worktree merge issue. Phase 28's type changes to 8 route files were computed against pre-Phase-27 versions, stripping the Zod query/param validation.
**Fix:** Restore `idParamSchema`, `booleanQueryParam`, and `dateRangeQuerySchema` validation in: agents.ts, products.ts, users.ts, alerts.ts, ai-budget.ts, call-audits.ts, sales.ts, call-logs.ts.
