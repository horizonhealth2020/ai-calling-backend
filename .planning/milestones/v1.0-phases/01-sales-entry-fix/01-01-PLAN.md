   ```typescript

   ```

   ```typescript
   setMsg({ text: `Failed to create sale (${res.status}): ${err.error ?? "Unknown error"}`, type: "error" });
   ```

4. **Update ALL other setMsg() calls in the file** to use the new typed format. Search for every `setMsg(` call and convert:
   - Success messages: `setMsg({ text: "...", type: "success" })`
   - Error messages: `setMsg({ text: "...", type: "error" })`

5. **Move the alert bar to ABOVE the form.** Currently the message display is below the submit button (~line 1132-1149). Move it to render above the sale entry form, inside the same container. Replace the existing fragile `msg.startsWith("Sale")` detection with the typed `msg.type` field.
