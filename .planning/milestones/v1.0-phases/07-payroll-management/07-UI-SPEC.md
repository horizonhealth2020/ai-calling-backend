## Color

All colors reference CSS custom properties from `packages/ui/src/theme.css`. Dark theme is default; light mode supported via `[data-theme="light"]`.

| Role | Token | Resolved (dark) | Usage in this phase |
|------|-------|-----------------|---------------------|
| Dominant (60%) | --bg-root | #070a0a | Page background |
| Secondary (30%) | --bg-surface-raised | #111c1c | AgentPayCard background |
| Accent (10%) | --primary-500 / --accent-teal | #14b8a6 | See reserved list below |
| Destructive | --danger | #f87171 | Unpaid indicator, late entry warning border |

### Accent reserved for:
- Top Earner badge border glow (rgba(20,184,166,0.25))
- Top Earner card background tint (rgba(20,184,166,0.04))
- Print button background and border
- Period selector active state

### Semantic colors used in this phase:
| Semantic | Token | Resolved (dark) | Usage |
|----------|-------|-----------------|-------|
| Success | --success | #34d399 | Paid button background tint, positive net amount, bonus highlight |
| Warning | --warning | #fbbf24 | Hold amount highlight, late entry "arrived after paid" indicator |
| Danger | --danger | #f87171 | Unpaid button, negative net amount, fronted highlight |
| Info | --info | #2dd4bf | Print button styling |

### Late entry indicator (Claude's discretion):
- Left border: 3px solid var(--warning) on entries detected as arrived-after-paid
- Background tint: rgba(251,191,36,0.04) on the entry row
- Tooltip or small label: "Arrived after paid" in 11px, color var(--warning)

### Paid card visual lockdown:
- Card opacity: 0.7 when all entries are PAID
- All input fields: `pointerEvents: "none"`, reduced opacity
- Financial summary inputs: `background: transparent`, no border, display as plain text
- Edit/Delete buttons: hidden (not rendered) on paid cards

## Focal Point

Primary focal point: the financial summary strip NET total value -- rendered at 16px/700 (Metric role), it carries the largest semantic weight in the AgentPayCard and is the first number a payroll operator scans to confirm correctness before marking paid.

## Component Inventory

### Modified Components

#### AgentPayCard (existing, modified)
Location: `apps/payroll-dashboard/app/page.tsx`

Changes:
1. **Collapsible entries**: Show first 5 entries. If entries.length > 5, render "Show {N} more" button below entry 5. On click, expand to show all entries. Show "Show less" to collapse.
2. **Paid lockdown**: When all entries have status PAID, disable all inputs (pointerEvents none), hide edit/delete actions, reduce card opacity to 0.7.
3. **Late entry indicator**: For entries where status is PENDING but sibling entries in the same agent+period have status PAID, render with warning left-border (3px solid var(--warning)) and "Arrived after paid" label.

#### Show More Button Style
```
background: var(--bg-surface-overlay)
border: 1px solid var(--border-default)
borderRadius: 6px (radius.md)
padding: 8px 16px
fontSize: 14px
fontWeight: 400
color: var(--text-secondary)
cursor: pointer
width: 100%
textAlign: center
transition: all 150ms ease-out
hover: background var(--bg-surface-raised), color var(--text-primary)
```

### No New Components
All UI changes are modifications to the existing AgentPayCard component. No new standalone components are introduced in this phase.

## Interaction States

### Collapsible Entries
| State | Visual |
|-------|--------|
| Collapsed (default, entries > 5) | First 5 entries visible. "Show {N} more" button below. |
| Expanded | All entries visible. "Show less" button below. |
| Collapsed (entries <= 5) | All entries visible. No button rendered. |

### Paid/Unpaid Toggle
| State | Visual |
|-------|--------|
| All entries PAID | Green "Paid" button (CheckCircle icon). Card at 0.7 opacity. Inputs disabled. |
| Not all entries PAID | Red "Unpaid" button (XCircle icon). Card at full opacity. Inputs enabled. |
| Toggle in progress | Button disabled, cursor wait. Restored on API response. |

### Late Entry Detection
| Condition | Visual |
|-----------|--------|
| Entry status PENDING + sibling entries PAID | Left border 3px solid var(--warning). Row background rgba(251,191,36,0.04). "Arrived after paid" label in 11px var(--warning). |
| Entry status PENDING + no sibling PAID | Normal entry rendering (no indicator). |
