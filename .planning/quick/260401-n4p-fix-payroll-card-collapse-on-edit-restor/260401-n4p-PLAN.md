**Bug 1 — Card collapse on edit:**

The `useEffect` at ~line 143 with dependency `[agentData]` calls `setExpandedAgents(new Set())` every time `agentData` changes. Since `agentData` is a `useMemo` depending on `periods`, any `refreshPeriods()` call (triggered by edit, approve, delete, mark paid, etc.) recalculates `agentData`, fires the effect, and collapses all agent cards.

Fix: Add a `useRef(false)` called `initializedRef`. In the useEffect:
- On first run (`!initializedRef.current`): set `initializedRef.current = true` and run the existing logic (collapse all agents, set default expanded weeks and selected weeks).
- On subsequent runs: do NOT call `setExpandedAgents(new Set())`. Instead, only update `expandedWeeks` and `selectedWeek` for agents that are NEW (not already in the maps). This preserves existing expand/collapse state while still initializing state for newly appeared agents.

```typescript
const initializedRef = useRef(false);

useEffect(() => {
  if (!initializedRef.current) {
    // First load: all agents collapsed, default weeks expanded
    initializedRef.current = true;
    setExpandedAgents(new Set());

    const weekMap = new Map<string, Set<string>>();
    const selMap = new Map<string, string>();
    for (const [agentName, data] of agentData) {
      const sorted = [...data.periods].sort((a, b) =>
        new Date(b.period.weekStart).getTime() - new Date(a.period.weekStart).getTime()
      );
      const expandedIds = new Set(sorted.slice(0, 2).map(p => p.period.id));
      weekMap.set(agentName, expandedIds);
      if (sorted.length > 0) selMap.set(agentName, sorted[0].period.id);
    }
    setExpandedWeeks(weekMap);
    setSelectedWeek(selMap);
    return;
  }

  // Subsequent updates: only add state for NEW agents, preserve existing
  setExpandedWeeks(prev => {
    const next = new Map(prev);
    for (const [agentName, data] of agentData) {
      if (!next.has(agentName)) {
        const sorted = [...data.periods].sort((a, b) =>
          new Date(b.period.weekStart).getTime() - new Date(a.period.weekStart).getTime()
        );
        next.set(agentName, new Set(sorted.slice(0, 2).map(p => p.period.id)));
      }
    }
    return next;
  });
  setSelectedWeek(prev => {
    const next = new Map(prev);
    for (const [agentName, data] of agentData) {
      if (!next.has(agentName)) {
        const sorted = [...data.periods].sort((a, b) =>
          new Date(b.period.weekStart).getTime() - new Date(a.period.weekStart).getTime()
        );
        if (sorted.length > 0) next.set(agentName, sorted[0].period.id);
      }
    }
    return next;
  });
}, [agentData]);
```
