## 1. Parser Logic (pure functions, defined above components)

**`parseChargebackText(raw: string): ParsedRow[]`**
- Split raw text by newline, filter empty lines
- For each line, split by tab character `\t`
- Row-number detection: if `fields[0]` matches `/^\d+$/` AND `fields[1]` matches a date pattern like `/\d{1,2}-[A-Za-z]{3}-\d{2}/`, shift fields array by 1 (skip the row number)
- Map tab positions to ParsedRow fields:
  - `fields[0]` -> postedDate (parse `DD-Mon-YY` format: split on `-`, map month name to number, assume 20xx for YY < 50)
  - `fields[1]` -> type (trim)
  - `fields[2]` -> payeeId (trim)
  - `fields[3]` -> payeeName (trim)
  - `fields[4]` -> payoutPercent (regex `/(\d+(?:\.\d+)?)%/` -> number)
  - `fields[5]` -> chargebackAmount (regex `/\(\$([\d,]+\.\d{2})\)/` -> negate, e.g. `($582.00)` -> `-582`)
  - `fields[6]` -> totalAmount (regex `/\$?([\d,]+\.\d{2})/` -> number)
  - `fields[7]` -> transactionDescription (trim)
  - `fields[8]` -> product (trim)
  - `fields[9]` -> member info pipe group: split by `|`, trim each. Index 0 = memberCompany, index 1 = memberId
  - `fields[10]` -> agent info pipe group: split by `|`, trim each. Index 0 = memberAgentCompany, index 1 = memberAgentId
- Return array of ParsedRow objects. Null for any field that fails to parse.

**`consolidateByMember(rows: ParsedRow[]): ConsolidatedRecord[]`**
- Group rows by `memberId ?? memberCompany ?? "unknown"` key
- For each group: take first row's postedDate/type/payeeId/payeeName/payoutPercent/transactionDescription/memberCompany/memberId/memberAgentCompany/memberAgentId
- Sum `chargebackAmount` across all rows in group
- Sum `totalAmount` across all rows in group
- Concatenate unique `product` values with `, ` separator
- Return array of ConsolidatedRecord with `assignedTo: ""` placeholder

**`assignRoundRobin(records: ConsolidatedRecord[], activeReps: string[]): ConsolidatedRecord[]`**
- If no active reps, return records with empty assignedTo
- Iterate records, assign `activeReps[(index) % activeReps.length]` to each record's `assignedTo`
- Return new array with assignments
