Add a new GET route `/ai/scoring-stats` to the existing ai-budget.ts router. The endpoint must:

1. **Auth:** `requireAuth, requireRole("OWNER_VIEW", "SUPER_ADMIN")` -- matches existing ai-budget.ts pattern.

2. **Date filtering:** Parse query params using `dateRange(req.query.range as string, req.query.from as string, req.query.to as string)` from `./helpers`. Import `dateRange` from `./helpers` (asyncHandler is already imported).

3. **Base where clause:** `const where: any = { aiScore: { not: null } };` -- CRITICAL: always filter out NULL aiScore records. If `dr` is defined, add `where.callDate = { gte: dr.gte, lt: dr.lt };`.

4. **Aggregate KPIs:** Use `prisma.callAudit.aggregate({ where, _avg: { aiScore: true }, _count: { id: true }, _min: { aiScore: true }, _max: { aiScore: true } })`.

5. **Score distribution:** Run 4 parallel count queries for buckets:
   - Poor (0-49): `{ ...where, aiScore: { not: null, gte: 0, lt: 50 } }`
   - Fair (50-69): `{ ...where, aiScore: { not: null, gte: 50, lt: 70 } }`
   - Good (70-84): `{ ...where, aiScore: { not: null, gte: 70, lt: 85 } }`
   - Excellent (85-100): `{ ...where, aiScore: { not: null, gte: 85, lte: 100 } }`
   Use `Promise.all` with `prisma.callAudit.count()` for each bucket. NOTE: Each bucket where clause must merge the date filter from `where` -- spread `where` first then override `aiScore` to include both `not: null` AND the range conditions.

6. **Per-agent breakdown:** Use `prisma.callAudit.groupBy({ by: ["agentId"], where, _avg: { aiScore: true }, _count: { id: true } })`. Then resolve agent names: `prisma.agent.findMany({ where: { id: { in: agentIds } }, select: { id: true, name: true } })`. Map to array of `{ agentId, agentName, avgScore: Math.round(avg ?? 0), auditCount }`.

7. **Weekly trends:** Fetch scored audits: `prisma.callAudit.findMany({ where, select: { aiScore: true, callDate: true }, orderBy: { callDate: "asc" } })`. Group by ISO week using UTC dates:
   ```typescript
   const weekMap = new Map<string, number[]>();
   for (const audit of scored) {
     const d = new Date(audit.callDate);
     const jan4 = new Date(Date.UTC(d.getUTCFullYear(), 0, 4));
     const dayOfYear = Math.floor((d.getTime() - new Date(Date.UTC(d.getUTCFullYear(), 0, 1)).getTime()) / 86400000);
     const weekNum = Math.ceil((dayOfYear + new Date(Date.UTC(d.getUTCFullYear(), 0, 1)).getUTCDay() + 1) / 7);
     const key = `${d.getUTCFullYear()}-W${String(weekNum).padStart(2, "0")}`;
     if (!weekMap.has(key)) weekMap.set(key, []);
     weekMap.get(key)!.push(audit.aiScore!);
   }
   ```
   Compute weekly averages, sort by week key ascending. For each week compute `delta` (difference from previous week's average, null for first week).

8. **Response shape:**

   ```json
     "aggregate": { "avgScore": 72, "totalAudits": 150, "minScore": 12, "maxScore": 98 },

   }
   ```
