**PART B: Role selector collapse delay (FIX-01)**

**1.** In `layout.tsx`, in the `DashboardInner` function (around line 61-73), modify the hover state logic:

FROM:
```typescript
const [hovered, setHovered] = useState(false);
// ...

```

TO:
```typescript
const [hovered, setHovered] = useState(false);
const [delayedHovered, setDelayedHovered] = useState(false);

useEffect(() => {
  if (hovered) {
    setDelayedHovered(true);
  } else {
    const timer = setTimeout(() => setDelayedHovered(false), 400);
    return () => clearTimeout(timer);
  }
}, [hovered]);

const expanded = delayedHovered || !activeTab;
```

**2.** Add `useEffect` to the React import if not already imported:
```typescript
import { useState, useEffect } from "react";
```
