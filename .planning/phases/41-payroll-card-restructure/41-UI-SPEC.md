## Interaction Contracts

### Agent Card Expand/Collapse (D-16)

| Property | Value |
|----------|-------|
| Trigger | Click anywhere on agent header row |
| Default state | All agent cards start expanded |
| Animation | Chevron rotates 180deg over 150ms with `cubic-bezier(0.16, 1, 0.3, 1)` |
| Content reveal | CSS class `animate-slide-down` (existing pattern from line 1847) |
| Multi-select | Multiple agent cards can be open simultaneously (Set-based state) |

### Week Section Expand/Collapse (D-14, D-15)

| Property | Value |
|----------|-------|
| Trigger | Click anywhere on week section header row |
| Default state | Last 2 weeks (most recent periods) expanded; older weeks collapsed |
| Animation | Same chevron rotation as agent card (14px icon, 150ms, ease-out) |
| Content reveal | Same `animate-slide-down` pattern |
| Multi-select | Multiple weeks can be open within the same agent card |

### Selected Week Indicator (D-05, D-06)

| Property | Value |
|----------|-------|
| Trigger | Click on a week section header to select it |
| Default | Most recent period is selected per agent |
| Visual indicator | 3px left border in `accentTeal` on selected week section |
| Background | `rgba(20,184,166,0.03)` on selected week section |
| Transition | `all 150ms ease-out` |
| Effect | Agent header read-only summary updates to show selected week's financials |

### Financial Strip Inputs (D-10, same as Phase 40)

| Property | Value |
|----------|-------|
| Input style | `SMALL_INP` (13px font, 90px width, right-aligned, 6px/10px padding) |
| Save trigger | `onBlur` fires `handleHeaderBlur()` which PATCHes AgentPeriodAdjustment |
| Disabled state | All inputs disabled + transparent background when all entries are paid |
| Live net update | Net recalculates on every input change (Commission + Bonus + Fronted - Hold) |

### Print Button (D-19)

| Property | Value |
|----------|-------|
| Location | Per-week, inside week section header area (right-aligned with Paid button) |
| Style | Ghost button with info-blue treatment: `background: infoBg`, `border: 1px solid rgba(45,212,191,0.2)`, `color: info` |
| Icon | `Printer` at 11px |
| Behavior | Opens print window for that single agent + single week's entries |

### Paid/Unpaid Button (D-12)

| Property | Value |
|----------|-------|
| Location | Per-week, next to print button in week section |
| States | Unpaid (red ghost), Mark Unpaid (green ghost with CheckCircle), Paid disabled (green ghost, reduced opacity, cursor: not-allowed when period locked) |
| Behavior | Unchanged from current AgentPayCard behavior |

## Layout Specifications

### Agent Card Wrapper

```
border: 1px solid borderDefault
border-radius: radius.xl (12px)
background: bgSurface (#0c1414)
margin-bottom: S[8] (32px) between agent cards
overflow: hidden (clip inner content to border-radius)
opacity: 0.7 when all entries across all weeks are paid
```

### Agent Header (inside agent card)

```
display: flex
justify-content: space-between
align-items: center
padding: S[4] (16px) vertical, S[5] (20px) horizontal
border-bottom: 1px solid borderSubtle
background: rgba(20,184,166,0.04) if top earner, transparent otherwise
cursor: pointer

Left side: agent name (17px/800) + Top Earner badge + sale count (12px)
Right side: read-only financial summary (13px) + chevron (18px)
```

### Week Section Header (inside agent card body)

```
display: flex
justify-content: space-between
align-items: center
padding: S[3] (12px) vertical, S[5] (20px) horizontal
cursor: pointer
border-bottom: 1px solid borderSubtle (when expanded)
background: rgba(255,255,255,0.02)

Left side: date range (14px/700) + status badge (Open/Locked)
Right side: Print button + Paid/Unpaid button + chevron (14px)
```

### Week Section Body (expanded content)

```
Same as current AgentPayCard body:
  Financial strip: flex, gap 16px, padding 10px 20px, background rgba(255,255,255,0.02)
  Sale table: full-width table with existing thStyle/tdStyle
  Pending approvals: below sale table (same as current)
```

### Selected Week Visual Treatment

```
border-left: 3px solid accentTeal (selected) / 3px solid transparent (not selected)
background: rgba(20,184,166,0.03) (selected) / transparent (not selected)
transition: all 150ms ease-out
```

### Week Section Divider (between weeks inside one agent card)

```
No explicit divider element needed.
Each week section has its own border-bottom on the header.
When collapsed, the week headers stack as a natural visual list.
Background treatment (rgba(255,255,255,0.02)) on week headers provides separation.
```

## Print Template Contract (D-19, D-20)

| Element | Treatment |
|---------|-----------|

| Week date range | Below agent name: "Week of Mar 24 - Mar 30" |

| Sale table | Agent, Status, Member, Product, Enroll Fee, Commission columns |
