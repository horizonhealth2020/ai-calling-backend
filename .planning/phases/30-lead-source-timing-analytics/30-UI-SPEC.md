## Typography

Uses existing tokens from `packages/ui/src/tokens.ts`. Phase-specific role mapping:

| Role | Size | Weight | Line Height | Usage in this phase |
|------|------|--------|-------------|---------------------|
| Table header (xs) | 11px | 600 (semibold) | 1.45 | Heatmap hour labels, sparklines table headers, axis labels |
| Body (sm) | 13px | 400 (normal) | 1.5 | Tooltip text, sparklines table cells, date filter labels |
| Card value (md) | 16px | 600 (semibold) | 1.5 | Best Source card close rate, lead source name in card |
| Section heading (lg) | 18px | 600 (semibold) | 1.4 | "Lead Source Timing Analytics" collapsible header |

All labels use `textTransform: "uppercase"` with `letterSpacing: typography.tracking.caps` ("0.06em") per existing `baseLabelStyle` / `baseThStyle` pattern. The combination of uppercase + letter-spacing + semibold (600) provides sufficient visual differentiation for header/label roles without requiring a third weight.

## Color

Uses existing CSS custom properties from `packages/ui/src/theme.css`. No new color tokens.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--bg-root` (#070a0a) | Page background behind analytics section |
| Secondary (30%) | `--bg-surface` (#0c1414) | LeadTimingSection wrapper, sparklines table background |
| Accent (10%) | `--accent-teal` (#14b8a6) | Collapsible header chevron on expand, sparkline polyline stroke, "Best Source" card border highlight, trend arrow (up) |
| Destructive | `--danger` (#f87171) | Trend arrow (down) only |

Accent reserved for: collapsible toggle icon (expanded state), sparkline SVG stroke, Best Source card left border accent, upward trend arrow icon.

### Heatmap Color Scale (phase-specific)

Diverging scale for close rate visualization on dark background. These are direct rgba values (not tokens) because they represent data-encoding, not UI chrome:

| Close Rate | Color | Opacity | Condition |
|------------|-------|---------|-----------|
| No data (0 calls) | `rgba(255,255,255,0.03)` | 1.0 | Empty cell -- distinguishable from zero conversions |
| 0% - 4.99% | `rgba(239,68,68,0.25)` | 1.0 | Low performance (red) |
| 5% - 9.99% | `rgba(234,179,8,0.30)` | 1.0 | Mid-low performance (yellow) |
| 10% - 14.99% | `rgba(34,197,94,0.30)` | 1.0 | Mid-high performance (green) |
| 15%+ | `rgba(34,197,94,0.50)` | 1.0 | High performance (bright green) |
| Any rate, < 10 calls | Same color as above | 0.3 | Low sample size de-emphasis per D-03 |

Business hour emphasis: Columns for hours 8-20 get `borderBottom: 1px solid rgba(255,255,255,0.06)` (slightly brighter than off-hours) to subtly highlight the meaningful range.

## Interaction Contracts

### Collapsible Section Toggle

| Trigger | Action |
|---------|--------|
| Click header row | Toggle `expanded` state |
| Expand | Render date filter + all child components, rotate chevron 180 degrees |
| Collapse | Hide all child components, chevron returns to 0 degrees |
| Transition | `max-height` + `opacity` transition at 250ms ease-out, or conditional render with no animation (simpler, matches existing patterns) |

### Heatmap Cell Hover

| Trigger | Action |
|---------|--------|
| mouseEnter on cell | Show tooltip positioned above cell (or below if near top edge) |
| mouseLeave on cell | Hide tooltip |
| Tooltip positioning | Absolute within heatmap container, offset 8px above cell center |

### Grouping Dropdown

| Trigger | Action |
|---------|--------|
| Change select value | Re-fetch heatmap data with new `groupBy` param (dow / wom / moy) |
| Loading during fetch | Existing heatmap cells show 0.3 opacity until new data arrives |

### Date Range Filter

| Trigger | Action |
|---------|--------|
| Select preset or custom range | Re-fetch all three endpoints (heatmap, sparklines, recommendation) with new date params |
| Default | "Last 30 Days" preset selected on first expand |

### Trend Arrow

| Trigger | Action |
|---------|--------|
| Positive delta (> 0%) | Green `TrendingUp` icon + "+X.X%" text |
| Negative delta (< 0%) | Red `TrendingDown` icon + "-X.X%" text |
| Flat (0% or insufficient prior data) | No arrow displayed |

## State Matrix

|-------|---------------|---------|------------|

## Responsive Behavior

This phase targets the existing ops-dashboard which is desktop-only (manager/owner workstations). No mobile breakpoints are required.

| Viewport | Behavior |
|----------|----------|
| >= 1280px | Full 24-column heatmap grid, all components at declared sizes |
| 1024-1279px | Heatmap cells shrink proportionally via `1fr` columns; minimum 28px cell width |
| < 1024px | Not a target viewport. Horizontal scroll on heatmap grid if needed (`overflow-x: auto`) |

---

## Accessibility Notes

| Concern | Approach |
|---------|----------|
| Heatmap color-only encoding | Tooltip on hover provides exact numeric values (rate, count) -- color is supplementary |
| Low sample de-emphasis | Opacity 0.3 + "(low sample size)" tooltip addendum |
| Keyboard navigation | Heatmap cells are not individually focusable (data visualization, not interactive controls); section toggle and dropdown are standard focusable elements |
| Screen reader | Section header uses semantic `<h3>`; heatmap uses `role="grid"` with `aria-label="Close rate heatmap by lead source and hour"` |
