<objective>
Sub-feature 1: Add a third commission rate `acaBundledCommission` on ADDON and AD_D products that applies when the sale is bundled with an ACA PL sale (sale.acaCoveringSaleId is set).

Output: Schema column, migration + prisma generate, form UI, API acceptance, calculateCommission wire-up.
</objective>
