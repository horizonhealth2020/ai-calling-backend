**Style constants (CSSProperties objects):**
- `SIDEBAR`: `{ width: 220, flexShrink: 0, borderRight: "1px solid " + C.borderSubtle, overflowY: "auto" as const, background: C.bgSurfaceRaised, display: "flex", flexDirection: "column" as const }`
- `SEARCH_WRAP`: `{ padding: S[3] + "px", borderBottom: "1px solid " + C.borderSubtle }`
- `SEARCH_INPUT`: spread `baseInputStyle` with overrides: `{ width: "100%", boxSizing: "border-box" as const, fontSize: 13, padding: "8px 12px 8px 32px" }` (left padding for icon)
- `SEARCH_ICON`: `{ position: "absolute" as const, left: S[3] + 10, top: "50%", transform: "translateY(-50%)", color: C.textMuted, pointerEvents: "none" as const }`
- `SECTION_HEADER`: `{ fontSize: 11, fontWeight: 600, textTransform: "uppercase" as const, letterSpacing: "0.06em", color: C.textTertiary, padding: S[2] + "px " + S[4] + "px", marginTop: S[2] }`
- `ROW` base: `{ display: "flex", alignItems: "center", height: 40, padding: "0 " + S[4] + "px", cursor: "pointer", transition: "background " + motion.duration.fast + " " + motion.easing.out, gap: S[1] }`
- `ROW_SELECTED` extends ROW: adds `background: C.bgSurfaceOverlay, borderLeft: "3px solid " + C.accentTeal, paddingLeft: S[4] - 3` (compensate for border)
- `TOP3_DOT`: `{ width: 6, height: 6, borderRadius: "50%", background: C.accentTeal, flexShrink: 0 }`
- `NAME_STYLE` (has sales): `{ fontSize: 13, fontWeight: 600, color: C.textSecondary, flex: 1, overflow: "hidden", textOverflow: "ellipsis", whiteSpace: "nowrap" as const }`
- `NAME_MUTED` (zero sales): same as NAME_STYLE but `color: C.textMuted`
- `EARNINGS_STYLE`: `{ fontSize: 13, fontWeight: 400, color: C.textTertiary, flexShrink: 0 }`
- `DIVIDER`: `{ height: 1, background: C.borderSubtle, margin: S[2] + "px 0" }`
- `NO_RESULTS`: `{ padding: S[4] + "px", fontSize: 13, color: C.textMuted, textAlign: "center" as const }`

**Component body:**
1. Internal state: `const [searchQuery, setSearchQuery] = useState("")`
2. Filter both `salesAgents` and `csAgents` by `searchQuery` using case-insensitive `includes()` on `agentName`
3. Render sidebar container with `SIDEBAR` style
4. Search input section:
   - Wrapper div with `SEARCH_WRAP` style and `position: "relative"`
   - `<Search size={14} />` icon with `SEARCH_ICON` style
   - `<input>` with `SEARCH_INPUT` style, placeholder `"Search agents..."`, value `searchQuery`, onChange updates state
5. Scrollable agent list (`flex: 1, overflowY: "auto"`)
6. Section header: "SALES AGENTS" with `SECTION_HEADER` style
7. Map filtered sales agents to rows:
   - Use `ROW_SELECTED` when `selectedAgent === agent.agentName`, else `ROW` with hover via onMouseEnter/Leave toggling a local hover state, OR simpler: just use inline `:hover` won't work with CSSProperties, so add `onMouseEnter` setting bg to `C.bgSurfaceOverlay` and `onMouseLeave` resetting (only when not selected)
   - Actually simpler approach: use a single `AgentRow` render function or inline. Selected rows get `ROW_SELECTED`. Non-selected rows get `ROW` base style. For hover, add `onMouseEnter` / `onMouseLeave` that set inline background (use a hovered state `Set` or just let CSS handle it via className — but project uses inline styles). Simplest: wrap each row in a div with the style, and use a helper that returns the correct style. Skip hover animation for simplicity (Claude's discretion allows instant transitions).
   - Each row contains:
     - If `agent.isTopEarner`: `<div style={TOP3_DOT} />`
     - Agent name span with `NAME_STYLE` (or `NAME_MUTED` if `agent.activeCount === 0`)
     - If `agent.activeCount > 0`: earnings span `formatDollar(agent.gross)` with `EARNINGS_STYLE`
     - If `agent.status !== null`: `<Badge color={badgeColor} size="sm">{badgeLabel}</Badge>` where:
       - `"paid"` -> `color: C.success`, label: `"Paid"`
       - `"unpaid"` -> `color: C.danger`, label: `"Unpaid"`
       - `"partial"` -> `color: C.warning`, label: `"Partial"`
   - `onClick={() => onSelectAgent(agent.agentName)}`
8. Divider div with `DIVIDER` style
9. Section header: "CUSTOMER SERVICE" with `SECTION_HEADER` style
10. Map filtered CS agents to rows (same row rendering as sales agents)
11. If both filtered lists are empty: show `<div style={NO_RESULTS}>No agents match your search.</div>`

**Export:** `export function AgentSidebar({ salesAgents, csAgents, selectedAgent, onSelectAgent }: AgentSidebarProps)`
