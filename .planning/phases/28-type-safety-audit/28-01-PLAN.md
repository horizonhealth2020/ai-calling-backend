**Category 2: `(req as any).user` access (alerts.ts:2, chargebacks.ts:1, pending-terms.ts:1)**

**Category 3: Dynamic Prisma where clauses (ai-budget.ts:1, call-audits.ts:3, sales.ts:1)**

- `ai-budget.ts`: `const where: Prisma.AiBudgetWhereInput = { ... }`

- `sales.ts`: `const where: Prisma.SaleWhereInput = { ... }`
Import `Prisma` from `@prisma/client` if not already imported.

**Category 4: Prisma JSON field casts (callAudit.ts:8, convosoKpiPoller.ts:1)**
